# KU PYP Management

A community-driven platform designed to help students at Kashmir University easily access and share previous year question papers.

## Features
- **Upload Papers**: Students can easily upload images of past question papers to contribute to the community.
- **Search & Download**: Filter papers by Semester, Subject, and Course Code to quickly find what you need.
- **Mobile-Responsive**: Clean, modern interface that works perfectly on phones, tablets, and desktops.
- **Dark/Light Mode**: Toggle between light and dark themes for comfortable reading.

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS (Custom Design System), Vanilla JavaScript.
- **Backend**: PHP 8.x, MySQL.
- **Icons & Tools**: Feather Icons, Cropper.js for image cropping.

## Setup Instructions
1. Clone the repository into your XAMPP `htdocs` folder.
2. Start Apache and MySQL from the XAMPP Control Panel.
3. Open `backend/db.example.php`, fill in your MySQL credentials (if not default), and rename the file to `db.php`.
4. Navigate to `http://localhost/ku-pyp%20management/backend/setup.php` in your browser to automatically initialize the database and tables.
5. Navigate to `http://localhost/ku-pyp%20management/` to start using the app.
